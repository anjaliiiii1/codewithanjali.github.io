1. What is a Variable?

Variables are used to store data values in a program. For example:
Python: x = 5
C: int x = 5;

2. Understanding Data Types

Different data types include:
- Integer (e.g., 10)
- Float (e.g., 3.14)
- String (e.g., "Hello")
- Boolean (True/False)

3. If-Else Conditions

Used to make decisions in a program.
Python: if age >= 18: print("Adult")

4. Loops

Loops let you repeat a block of code:
Python: for i in range(5): print(i)

5. Functions

Functions group reusable code:
Python: def greet(): print("Hello")
