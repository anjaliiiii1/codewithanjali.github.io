HTML Images

To add images in HTML, we use the <img> tag. It is a self-closing tag and requires the src and alt attributes.

🔍 Example:

<img src="profile.jpg" alt="Profile Photo" width="300" height="200">

📝 Attributes:

  • src: Path to the image file.
  • alt: Alternative text for screen readers or when the image doesn't load.
  • width and height: Dimensions of the image (optional).

📌 Best Practices:

  • Always add alt text for accessibility.
  • Use optimized image sizes for faster loading.
