HTML Attributes

Attributes provide extra information about HTML elements. They are always included in the opening tag and usually come in name/value pairs like name="value".

🔍 Example:

<a href="https://www.codewithanjalii.com">Visit My Website</a>

In this example, href is the attribute name, and the link is the value.

✨ Common HTML Attributes:

  • href – URL for links
  • src – Image source
  • alt – Alternative text for images
  • title – Tooltip text
  • style – Inline CSS styling
  • id – Unique identifier
  • class – CSS class name
