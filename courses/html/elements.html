HTML Elements

HTML Elements are the building blocks of webpages. An element includes the start tag, content, and the end tag.

🔍 Example:

<p>This is a paragraph element.</p>

Here, <p> is the **start tag**, "This is a paragraph element." is the **content**, and </p> is the **end tag**.

📚 Types of Elements:

  • Block-level: Start on a new line. Example: <div>, <p>, <h1>
  • Inline: Stay within the line. Example: <span>, <a>, <strong>
  • Empty Elements: Don’t need closing tag. Example: <br>, <img>
