HTML Paragraphs

Paragraphs in HTML are created using the <p> tag. They help structure text content clearly and readably.

🔍 Example:

<p>This is a paragraph in HTML.</p>
<p>Each paragraph starts on a new line.</p>

Browsers automatically add space before and after each paragraph for better readability.

📌 Best Practices:

  • Use <p> for blocks of text, not just single lines.
  • Avoid using multiple <br> tags to create paragraphs — use <p> instead.
